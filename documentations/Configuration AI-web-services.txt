### Production Deployment

1. **Set GitHub Secrets**:
   - `DOCKER_USERNAME`
   - `DOCKER_PASSWORD`
   - `PRODUCTION_HOST`
   - `PRODUCTION_USER`
   - `SSH_PRIVATE_KEY`

2. **Configure Domain**:
   - Update `nginx.conf` with your domain
   - Set up SSL certificates

3. **Deploy**:
   ```bash
   git push origin main
   ```
